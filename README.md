## Game Overview

上から落ちてくるゴミをプレイヤーが操作して、
正しい場所へ導くアクションゲームです。

シンプルな操作ながら、判断力と反射神経が求められます。


## How To Play

キーボードでは、← →の矢印キーか、A Dキーで、タッチパネルでは、画面の右側と左側をタップすることで、操作が可能です。

また、escキーもしくはXキーで、ゲーム中メニューが開く、Rでリトライ、Tでタイトル推移などのショートカットを用意しています。
