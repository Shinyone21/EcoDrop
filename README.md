# EcoDrop
学校の研究の一環で作成した、2D思考型アクションゲーム「EcoDrop」の公開ページです。

## Play in your Browser Online

→ https://shinyone21.github.io/EcoDrop/

### Supported Browsers

- Chrome
- Firefox
- Safari

PCだけでなく、スマートフォンやタブレットでも動作しますが、一部端末では動作が不安定になる場合があります。

そのような場合、ダウンロード版を推奨します。

## Game Overview

上から落ちてくるゴミをプレイヤーが操作して、
正しいゴミ箱へ導くアクションゲームです。

シンプルな操作ながら、判断力と反射神経が求められます。


## How To Play

キーボードでは、← →の矢印キーか、A Dキーで、タッチパネルでは、画面の右側と左側をタップすることで、操作が可能です。

また、Xキーで、ゲーム中メニューが開く、Rでリトライ、Tでタイトル推移などのショートカットを用意しています。
