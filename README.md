# EcoDrop

ブラウザで遊べる3Dアクションゲーム「EcoDrop」の公開ページです。

## 🎮 Play Now

👉 https://shinyone21.github.io/EcoDrop/

※PCブラウザ推奨ですが、スマートフォンでも動作します。画面を横にしてください。

---

## 🧩 Game Overview

上から落ちてくるオブジェクトを操作して、
正しい場所へ導くアクションゲームです。

シンプルな操作ながら、判断力と反射神経が求められます。
